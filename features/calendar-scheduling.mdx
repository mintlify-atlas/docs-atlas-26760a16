## Creating Bookings from Calendar

Add appointments directly from the calendar interface:

<Steps>
  <Step title="Click Create Menu">
    Click the "+" button in the calendar toolbar
  </Step>
  
  <Step title="Choose Booking Type">
    Select:
    - **New Booking**: Customer appointment
    - **Blocked Time**: Mark time as unavailable
  </Step>
  
  <Step title="Fill Details">
    Side panel opens with the booking form (see [Booking Management](/features/booking-management))
  </Step>
  
  <Step title="Save">
    Booking appears immediately in the calendar
  </Step>
</Steps>

## Drag-and-Drop Rescheduling

<Info>
  Quickly reschedule appointments by dragging bookings to new time slots.
</Info>

### How It Works

1. **Click and hold** any booking event
2. **Drag** to the desired new time slot
3. **Drop** the booking
4. **Confirm** the reschedule in the dialog

<img src="/images/drag-booking.png" alt="Drag and Drop Booking" />

### Drag Restrictions

<Warning>
  You cannot reschedule bookings to times in the past. The calendar automatically prevents this.
</Warning>

The system validates:
- New time must be in the future
- Booking duration stays the same
- New time must fit within a single day (no overnight drag)

### Confirmation Dialog

Before finalizing the reschedule, you'll see:
- Original date and time
- New date and time
- Option to confirm or cancel
- Ability to revert by clicking cancel

<Accordion title="Canceling a Drag">
  
  If you change your mind:
  1. Click "Cancel" in the confirmation dialog
  2. The booking returns to its original time
  3. No changes are saved
  
</Accordion>

## Blocked Time Management

Mark time as unavailable for bookings:

### Creating Blocked Time

<Steps>
  <Step title="Open Create Menu">
    Click "+" button → "Block Time"
  </Step>
  
  <Step title="Configure Block">
    Set:
    - Block type (vacation, meeting, break, etc.)
    - Start and end time
    - All-day option
    - Employee assignment (if applicable)
  </Step>
  
  <Step title="Add Icon (Optional)">
    Choose an emoji or icon to identify the block type
  </Step>
  
  <Step title="Save">
    Time is marked as unavailable
  </Step>
</Steps>

### Blocked Time Display

<CardGroup cols={2}>
  <Card title="Visual Style" icon="ban">
    - Dark semi-transparent background
    - Clearly distinguishable from bookings
    - Shows block name and icon
  </Card>
  
  <Card title="Behavior" icon="lock">
    - Prevents customer bookings in that time
    - Can be edited or deleted
    - Can be moved by dragging
    - Supports all-day blocks
  </Card>
</CardGroup>

### Editing Blocked Time

Click any blocked time block to:
- View block details
- Change duration or time
- Update block type
- Delete the block
